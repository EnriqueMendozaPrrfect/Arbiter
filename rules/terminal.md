# Terminal Rules

## 1. On Execution

- Always ask for confirmation before executing any command.
- Always explain what the command does before executing it.
- Always show the output of the command after executing it.

## 2. On Error

- Always show the error message.
- Always explain what the error means.
- Always suggest a solution.

## 3. On using ports

- Always check if the port is already in use.
- Always ask for confirmation before using a port.
- Always show the output of the command after executing it.
- Always kill the process if it is not in use.
